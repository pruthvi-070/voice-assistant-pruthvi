# Voice Assistant - Pruthvi

This is a Python-based voice assistant.

## Features
- Voice commands 🎤
- Text-to-speech 🔊
- Task management 🧠
- Screenshot capture 📸
- Open browser 🌐

## How to Run
1. Install dependencies:
   pip install SpeechRecognition gTTS pyautogui pydub pyaudio

2. Run:
   python assistant.py

3. Say:
   pruthvi add a task
